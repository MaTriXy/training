XML and JSON

XML Parsing Options

  • DOM
  • SAX
  • XmlPullParser

Document Object Model

  • They load the entire document
  • You have random access to the content
  • With limited resources is not the optimal solution
  • You can randomly access with queries the content

Stream based Parsers

  • They process the document while reading the data source
  • Push parses : callbacks (sax)
  • Pull parses : are more like iterators or cursors (client ask for the next element to be retrieved)
  • Lack of internal data structure to help you in the navigation

JSON Parsing Options

  • JSON
  • Jackson
  • Google's gson

JSON

  • Android comes with the JSON implementation from json.org
  • JSON load the entire document
  • If the json is too big try with gson stream or jackson

JSON example

String content = "[{\"color\":\"red\",\"type\":\"ferrari\"}, {...}]";
JSONArray jsonArray = new JSONArray(content);

//you can iterate through the array to get object
JSONObject jsonCar = jsonArray.getJSONObject(0);

//getContent of the object
jsonCar.getString("colour");
jsonCar.getString("type");
