UI and design

UI

    As mentioned before, Android's UI is made up of layouts and widgets

    Configuring and combining these we come up with dialogs, menus, notifications...

    We configure layouts and widgets through attributes in xml files

<TextView
    android:id="@+id/signupnow_title"
    android:paddingBottom="@dimen/act_gutter_padding"
    android:text="@string/signupnow_title"
    android:textStyle="bold"
    android:textSize="@dimen/text_size_p1"
    android:textColor="@color/text_hint"
    android:gravity="center"
    android:visibility="gone" />

Application Resources

  • Always externalize resources such as images and strings from your application code, so that you can maintain them independently
  • Allows you to provide alternative resources that support specific device configurations such as different languages or screen sizes
  • Placed in the res folder of your project eg:
    • res/
    drawable/  
        icon.png  
    layout/  
        main.xml
        info.xml
    values/  
        strings.xml
